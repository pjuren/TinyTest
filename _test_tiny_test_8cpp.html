Detailed Description

This file defines a set of standard tests to run on the comparison macros that come with TinyTest, some of which are expected to succeed and some of which are expected to fail. The results can be compared against the file regressionTestExpectedOutput.txt using diff; this is a rudimentary regression test and is implemented in the included make file as the target 'test'.
